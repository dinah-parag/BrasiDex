# BrasiDex - Um Catálogo de Receitas Brasileiras

*Disponível em*: https://brasi-dex.vercel.app/

*O que é*: Um app que permite o usuário a pesquisa de receitas brasileiras entre mais de 8.000 receitas.

*Conceitos Teóricos*: v-for, Propriedades Computadas (Computed Properties - vital no Vue), Vue Router (navegação básica), Props e Emits.
